# Primitive トークン

生の値に名前をつけた基本トークン。プロジェクトに応じて定義する。

## 色

| トークン名 | 値 | 用途 |
| --- | --- | --- |
| `color-blue-500` | `#3B82F6` | 青系の基準色 |
| `color-gray-100` | `#F3F4F6` | 薄いグレー |
| `color-gray-900` | `#111827` | 濃いグレー |
| `color-white` | `#FFFFFF` | 白 |
| `color-red-500` | `#EF4444` | 赤系の基準色 |
| `color-green-500` | `#22C55E` | 緑系の基準色 |
