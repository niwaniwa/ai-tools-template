# Component トークン

Semantic トークンを特定のUIコンポーネントに紐づけたもの。

## ボタン

| トークン名 | 参照先 | 用途 |
| --- | --- | --- |
| `button-bg-primary` | `color-primary` | プライマリボタンの背景 |
| `button-text-primary` | `color-background` | プライマリボタンのテキスト |
| `button-padding-x` | `spacing-md` | ボタンの左右パディング |
| `button-padding-y` | `spacing-sm` | ボタンの上下パディング |
| `button-radius` | `radius-md` | ボタンの角丸 |

## カード

| トークン名 | 参照先 | 用途 |
| --- | --- | --- |
| `card-bg` | `color-surface` | カード背景 |
| `card-padding` | `spacing-md` | カード内余白 |
| `card-radius` | `radius-lg` | カード角丸 |

## インプット

| トークン名 | 参照先 | 用途 |
| --- | --- | --- |
| `input-bg` | `color-background` | 入力欄の背景 |
| `input-text` | `color-text` | 入力テキスト色 |
| `input-padding` | `spacing-sm` | 入力欄の内余白 |
| `input-radius` | `radius-md` | 入力欄の角丸 |
| `input-border-error` | `color-error` | エラー時のボーダー色 |
